# Lucas Language
An interpreted programming language written in Rust, with Brazilian Portuguese syntax.

## Installation

### Prerequisites

- Rust (v1.70 or higher)
- Cargo (comes with Rust)

### Build

```
git clone https://github.com/LucasHT22/lucas
```

```
cargo build
```

#### Run REPL

```
cargo run
```
